# Introducción

Este documento forma parte de la entrega final del curso de Bases de datos SQL de Coderhouse. En el presente proyecto me dediqué a diseñar el sistema de bases de datos para un modelo de videojuego multijugador masivo online de rol (MMORPG) ambientado en un mundo de fantasía medieval. Se modelaron sistemas de autentificación (username, email, password en texto plano), personajes, clases, estadísticas de combate, objetos, inventario, misiones, localizaciones, mascotas, entre otras entidades del mundo.
